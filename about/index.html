👨‍💻 关于我

大家好！我是 Tisha，一名拥有 4 年经验的软件测试工程师。💼

我的职业旅程从传统的手动测试开始，现在正逐步迈向测试开发的世界。通过不断学习与实践，我已经成功实现了一个 接口自动化测试平台，并计划在未来开发一个基于 Playwright 的 UI 自动化测试平台。除此之外，我还打算深入研究性能测试，并与大家分享相关的技术与经验。

🚀 职业目标

  • 构建强大且可扩展的自动化测试框架
  • 持续学习并掌握前沿的测试技术
  • 分享我的测试开发经验，助力更多人提升技能

🔧 目前正在做的项目

  • 接口自动化测试平台：已完成，并在多个项目中得到应用。✅
  • Playwright UI 自动化测试平台：开发中，敬请期待！🕒
  • 性能测试分享：探索性能测试工具和方法，计划与大家一起交流。📈

💡 技能树

  • 编程语言：Python 🐍
  • 测试工具：JMeter ⚙️
  • 框架：FastAPI ⚡
  • 前端框架：Vue 3 🌐，正在学习 React ⚛️

我致力于通过这些技能点，不断提升测试自动化的效率和质量。

🎨 兴趣爱好

工作之外，我是一个热爱生活的人。⛰️

  • 爬山：喜欢亲近自然，征服高峰，每次登顶都是一种成就感。🏞️
  • 唱歌：音乐是生活的调味品，尤其是当一首好歌在耳边响起时，我总是忍不住要跟着唱几句。🎤

希望通过这份简短的介绍，你们能更好地了解我。如果你也在探索测试开发的道路，或者对爬山和唱歌有共同的兴趣，欢迎和我交流！😊
